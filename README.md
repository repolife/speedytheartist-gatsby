# Start Here

### Set the following env varabiles in a .env.development file

-   SPOTIFY_CLIENT_ID
-   SPOTIFY_SECRET
-   GATSBY_SPOTIFY_ARTIST_ID

-   Install deps `npm ci`
-   Instal Netlify CLI `npm install netlify-cli -g`
-   Run locally `npm run develop`
-   Run with Netlify Functions `ntl dev`
